# Safe Reinforcement Learning with a Neural Safety Shield in GridWorld

This project implements a safe reinforcement learning workflow in a custom GridWorld environment. It combines tabular Q-learning, graph-based danger-map supervision, supervised neural safety classification, and shield-guided action intervention.

The main goal is to study how a learned safety shield changes exploration behaviour: whether it can reduce unsafe actions while still allowing the agent to learn useful policies.

## Project Overview

The workflow has four main stages:

1. **GridWorld environment design**
   A 10×10 grid environment is implemented with fixed hazard cells, a start state, a goal state, boundary handling, reward logic, and both fixed-start and random-start reset modes.

2. **Baseline Q-learning**
   A tabular Q-learning agent is trained without a safety shield to show how an unconstrained RL agent can learn successful policies while still producing unsafe transitions during exploration.

3. **Danger-map supervision and safety dataset generation**
   A multi-source breadth-first search is used to compute each cell's distance to the nearest hazard. This danger map is then used to label every valid state-action pair into four risk classes.

4. **Neural safety shield and Safe RL integration**
   A small neural network classifier is trained to predict the risk class of each state-action pair. The trained shield is integrated into the Q-learning loop to intervene when the proposed action is predicted to be unsafe.

## Key Features

* Custom 10×10 GridWorld environment
* Tabular Q-learning baseline agent
* Multi-source BFS danger-map construction
* Complete state-action safety dataset generation
* Four-class neural safety shield classifier
* Shield-guided action selection with configurable risk thresholds

## Methods

### 1. GridWorld Environment

The environment contains:

* 100 grid cells
* 15 hazard cells
* 4 discrete actions: up, down, left, right
* terminal goal and hazard states
* configurable step penalty
* fixed-start and random-start training modes

The environment returns structured transition information so that both task performance and safety violations can be tracked during training.

### 2. Baseline Q-learning

The baseline agent uses a tabular Q-value representation over state-action pairs. It selects actions with an epsilon-greedy policy and updates Q-values using the standard Q-learning update rule.

Two step-penalty settings are compared to understand how reward shaping affects learning progress, path efficiency, and unsafe exploration.

### 3. Danger Map and Safety Dataset

A multi-source BFS starts from all hazard cells and computes the minimum distance from every grid cell to the nearest hazard.

Each valid state-action pair is labelled into one of four risk classes:

| Class | Meaning               |
| ----- | --------------------- |
| 0     | Immediate hazard      |
| 1     | One step from hazard  |
| 2     | Two steps from hazard |
| 3     | Safe                  |

The resulting dataset contains 336 labelled state-action samples with 10-dimensional feature vectors.

### 4. Neural Safety Shield

The safety shield is a feed-forward neural network trained as a multi-class classifier.

Input features include:

* current position
* one-hot encoded action
* next position
* current distance to nearest hazard
* next-state distance to nearest hazard

The shield predicts the risk class for each candidate action before the RL agent executes it.

### 5. Shield-Guided Safe RL

During Safe RL training, the Q-learning agent first proposes an action. The safety shield then evaluates the proposed action and intervenes if its predicted risk class is above the configured risk threshold.

The project compares:

* no shield
* lenient shield threshold
* strict shield threshold

This allows the safety-performance trade-off to be evaluated directly.

## Results

### Baseline Q-learning

The baseline experiment compares two step-penalty settings under fixed-start training.

![Baseline Q-learning comparison](assets/baseline_q_learning_comparison.png)

In the saved run, the gentler step penalty achieved stronger final learning performance:

| Configuration  | Success Rate | Average Reward | Average Length | Total Violations |
| -------------- | -----------: | -------------: | -------------: | ---------------: |
| penalty = -1.0 |         0.0% |         -7.604 |          17.88 |              590 |
| penalty = -0.1 |        97.4% |          7.785 |          17.95 |              508 |

The result shows that baseline Q-learning can learn a successful policy, but it still accumulates many hazard violations during exploration.

### Danger Map

The danger map provides a compact safety signal by measuring each cell's distance to the nearest hazard.

![Danger map heatmap](assets/danger_map_heatmap.png)

This graph-based supervision is used to construct the full labelled safety dataset for shield training.

The neural safety shield learns to classify state-action risk levels from the generated dataset.

### Safe RL Threshold Comparison

The final experiment integrates the trained safety shield back into the Q-learning loop and compares different intervention thresholds.

![Safe RL threshold comparison](assets/safe_rl_threshold_comparison.png)

| Configuration | Success Rate | Average Reward | Total Violations | Avg. Interventions / Episode |
| ------------- | -----------: | -------------: | ---------------: | ---------------------------: |
| No shield     |        99.2% |          8.983 |              198 |                        0.000 |
| Theta = 0     |       100.0% |          9.155 |                0 |                        0.668 |
| Theta = 2     |         6.4% |        -18.083 |                0 |                      182.013 |

The lenient shield setting achieved the best balance in this environment: it removed safety violations while keeping the success rate and average reward high. The strict setting also avoided unsafe transitions, but it intervened too frequently and prevented efficient learning.

## Main Takeaways

* Baseline Q-learning can learn the task but does not guarantee safety during exploration.
* A danger map built from hazard locations provides a useful supervisory signal for safety classification.
* A compact neural shield can learn the state-action risk labels in this controlled GridWorld setting.
* Shield-guided action intervention can remove unsafe transitions.
* Safety thresholds create a clear trade-off between task performance, intervention frequency, and safety strictness.

## Limitations

This project uses a small, controlled GridWorld environment. The safety dataset is generated from a known hazard layout, so the shield is trained with complete environment-derived supervision. More realistic Safe RL settings would involve larger state spaces, partial observability, delayed feedback, stochastic transitions, or human intervention signals.

## Future Extensions

Possible extensions include:

* testing larger and procedurally generated environments
* adding stochastic transitions or noisy observations
* replacing tabular Q-learning with deep Q-learning
* training shields from sparse intervention data instead of complete danger-map labels
* evaluating false-positive and false-negative trade-offs under distribution shift
* comparing rule-based shields with learned neural shields
